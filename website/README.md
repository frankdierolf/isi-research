## Overview

A single-page Vue.js site presenting ISI Voice Image as an academic research project. Deployed at [isi-research.org](https://isi-research.org).

## The Concept

The website frames the hackathon project as a "pop-up research institute" — a playful academic presentation with research paper styling, download section, and about info.

## Tech Stack

- Vue.js 3 + TypeScript + Vite
- Deployed on GitHub Pages
- Custom domain: isi-research.org
